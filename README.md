# Vote no Restaurante

**Requisitos para Instalação**

 - Java 8
 - Maven 3.2.3 (+)

**O que foi utilizado**

 - VRaptor + JSP
 - CDI
 - jUnit
 - JPA + Hibernate
 - Bootstrap
 - jQuery

**Instruções para instalação**

 - Execute o comando `mvn clean compile wildfly:run` 
 - Acesse `127.0.0.1:8080`.
